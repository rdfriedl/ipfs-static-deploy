# Usage

<!-- usage -->

```sh-session
$ npm install -g ipfs-static-deploy
$ ipfs-static-deploy COMMAND
running command...
$ ipfs-static-deploy (-v|--version|version)
ipfs-static-deploy/0.1.0 linux-x64 node-v10.9.0
$ ipfs-static-deploy --help [COMMAND]
USAGE
  $ ipfs-static-deploy COMMAND
...
```
